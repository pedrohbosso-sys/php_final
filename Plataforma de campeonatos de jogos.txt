# ProLeague

### Plataforma de Campeonatos de Jogos

---

# 1. Descrição do Projeto

O projeto **ProLeague** será uma plataforma web de campeonatos de jogos eletrônicos desenvolvida utilizando:

* PHP
* HTML
* CSS
* JavaScript
* PostgreSQL

O sistema terá como objetivo permitir que usuários criem contas, participem de campeonatos, criem times e acompanhem rankings e partidas.

O projeto possuirá uma interface moderna com tema gamer, utilizando cores escuras, efeitos neon e layout responsivo.

---

# 2. Objetivo do Sistema

O sistema tem como objetivo:

* Permitir cadastro e login de usuários
* Permitir criação de times
* Exibir campeonatos disponíveis
* Gerenciar partidas
* Mostrar rankings dos times
* Possuir uma área administrativa simples
* Demonstrar o uso de banco de dados com PHP

# 6. Estrutura do Banco de Dados

## Tabela: usuarios

```sql
CREATE TABLE usuarios (
    id SERIAL PRIMARY KEY,
    nome VARCHAR(100),
    email VARCHAR(100),
    senha VARCHAR(255)
);
```

---

## Tabela: times

```sql
CREATE TABLE times (
    id SERIAL PRIMARY KEY,
    nome VARCHAR(100),
    jogo VARCHAR(50),
    descricao TEXT,
    usuario_id INT
);
```

---

## Tabela: campeonatos

```sql
CREATE TABLE campeonatos (
    id SERIAL PRIMARY KEY,
    nome VARCHAR(100),
    jogo VARCHAR(50),
    data_campeonato DATE,
    status VARCHAR(50)
);
```

---

## Tabela: partidas

```sql
CREATE TABLE partidas (
    id SERIAL PRIMARY KEY,
    time1_id INT,
    time2_id INT,
    placar1 INT,
    placar2 INT
);
```
